# Многофункциональный Telegram бот

Telegram бот, который умеет определять местоположение по IP-адресу и отвечать на вопросы.

## Описание

Бот обрабатывает текстовые сообщения двумя способами:
- Если запрос связан с IP-адресом (содержит IP, домен или ключевые слова), бот показывает геолокационную информацию
- В остальных случаях бот отвечает на сообщение как обычный собеседник

## Возможности

- Определение местоположения по IP-адресу
- Преобразование доменных имен в IP-адреса
- Получение информации о городе, регионе, стране и провайдере
- Автоматическое распознавание запросов об IP в сообщениях
- Ответы на общие вопросы и поддержка диалога

## Стек

- Python 3.10+
- python-telegram-bot (асинхронная версия)
- requests для выполнения HTTP-запросов
- python-dotenv для работы с переменными окружения
- socket для определения IP доменов
- Регулярные выражения для поиска шаблонов

## Установка

1. Клонируйте репозиторий:
git clone <repository-url>
cd Bot

2. Создать и активировать окружение:
python -m venv .venv
.venv\Scripts\activate

3. Создать файл .env в корне проекта
   
BOT_TOKEN=your_telegram_bot_token (Создать бота и взять токен с https://t.me/botfather)


GITHUB_TOKEN=your_github_token (Github токен для ai модели брать от сюда: https://github.com/settings/personal-access-tokens/)
